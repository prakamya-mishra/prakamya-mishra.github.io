Smart Luggage: Automatic Follower Bot, Anti-Theft, No GPS, and Real Time
Top 20 teams| Hack in the North 2018, IIIT Allahabad
Prakamya Mishra, Shashwat Kedia, Mudit Saexna
This project aims to make an Automatic Trolley Human Follower for a general or an industrial user that is affordable. The main objective is to reduce the cost yet increase the reliability, provide a viable solution for the old and the disabled. This automatic trolley human follower is controlled by an Arduino, that can follow the user automatically using a combination of ultrasonic range sensors, accelerometer, gyroscope, fingerprint sensor and camera.
